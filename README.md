# Sallio - Modern Functional Website

A fully functional, modern website built with HTML5, CSS3, and vanilla JavaScript.

## Features

### 🎨 Design & UI
- **Responsive Design**: Works seamlessly across desktop, tablet, and mobile devices
- **Modern Interface**: Clean, professional design with smooth animations
- **Accessibility**: Semantic HTML with proper ARIA labels
- **Color Scheme**: Professional blue theme with smooth transitions

### 🚀 Functionality

#### Navigation
- Sticky header that stays visible while scrolling
- Mobile-responsive hamburger menu
- Smooth scroll navigation to sections
- Active section highlighting in navigation

#### Interactive Elements
- **Hero Section**: Eye-catching hero with call-to-action button
- **Feature Cards**: Animated cards with hover effects
- **Statistics Counter**: Animated number counters that trigger on scroll
- **Contact Form**: Fully functional form with validation and feedback
- **Parallax Effects**: Subtle parallax scrolling on hero section

#### Form Features
- Real-time input validation
- Visual feedback (green/red borders)
- Form submission with success message
- Automatic form reset after submission

#### Animations
- Fade-in animations on page load
- Scroll-triggered animations for feature cards
- Smooth transitions on all interactive elements
- Hover effects on buttons and cards

## Technologies Used

- **HTML5**: Semantic markup with accessibility features
- **CSS3**: Modern styling with flexbox, grid, and animations
- **JavaScript**: Vanilla JS for all interactions (no frameworks)

## How to Use

1. Open `index.html` in any modern web browser
2. Navigate using the menu or scroll through sections
3. Try the interactive features:
   - Toggle mobile menu (resize browser to mobile width)
   - Scroll to see animations
   - Fill out the contact form
   - Hover over feature cards

## Sections

1. **Home/Hero**: Welcome section with call-to-action
2. **Features**: Four feature cards showcasing capabilities
3. **About**: Statistics section with animated counters
4. **Contact**: Functional contact form with validation
5. **Footer**: Social links and copyright information
